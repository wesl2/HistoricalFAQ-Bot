### 6. 对话历史格式变更

**旧版（自定义格式）**：
```sql
CREATE TABLE chat_history (
    id SERIAL PRIMARY KEY,
    session_id VARCHAR(100),
    user_query TEXT,
    retrieved_faq_ids INTEGER[],
    retrieved_doc_ids INTEGER[],
    llm_response TEXT,
    llm_mode VARCHAR(20),
    created_at TIMESTAMP
);
```

**新版（LangChain 标准）**：
```sql
CREATE TABLE chat_history (
    id SERIAL PRIMARY KEY,
    session_id VARCHAR(100) NOT NULL,
    role VARCHAR(20) NOT NULL,        -- 'human' 或 'ai'
    content TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

**优势**：兼容 `PostgresChatMessageHistory`，与 LangChain 生态无缝集成。

1. **代码与教程同步**：当前教程已更新至最新代码状态，可直接参考实现。

3. **依赖安装**：标准架构需要安装：
```bash
pip install langchain-openai langchain-huggingface
```

4. **环境变量配置**：使用 API 模式时，需要正确配置环境变量：
```bash
export API_PROVIDER=deepseek  # 或 openai、moonshot、qwen
export API_KEY=your-api-key
export API_BASE_URL=https://api.deepseek.com/v1
export API_MODEL=deepseek-chat
```
